# 旅行お助け君（フロント）

## 開発環境構築

### bun のインストール

https://bun.sh/docs/installation

### node ライブラリインストール

```sh
bun install
```

### dev サーバー起動

```sh
bun run dev
```

## build + preview

```sh
bun run build
bun run preview
```

## lint + format

```sh
bun run check
```

## サンプルページの閲覧方法

```sh
bun run build
bun run preview
```

o
